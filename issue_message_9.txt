3. 세번째로 Home page banner의 text하고 button이 위로 쏠린 문제를 해결했습니다.

---------------------

4. 네번째로 Home page banner의 다음 버튼을 누르면 완벽하게 target section으로 갈 수 있게 해결했습니다.

---------------------

5. 다섯번째로 Home page의 콘텐츠가 mobile, tablet, 그리고 desktop에서 깔끔하게 보이도록 layout을 정리 했습니다.

---------------------

6. 여섯번째로 blog page, project page, document page 안에 있는 side bar menu가 mobile header menu 안에 끼는 문제를 해결했습니다.

7. 일곱번째로 hovoer color를 sk telecom theme에 맞게 정리 했습니다.

---------------------
